Access Profile User Manager

Description: Access User Manager is an Odoo application designed to streamline and simplify user access management.
It provides an intuitive interface for users to control various access rights without requiring intricate technical knowledge.

With this tool, users can hide menus, buttons, and tabs, manage model access rights (including create, edit, delete, view, actions, archive/unarchive, duplicate, and export), and set field access rights (such as making fields invisible, required, or read-only).
Additionally, the application allows for domain-based access rights, enabling conditional restrictions on creating, editing, deleting, and reading records based on field values.
Users can hide specific buttons, tabs, or menu/submenu items and make any user read-only, hide chatter, or disable developer mode.
Model-wise chatter access rights and global access rights for chatter items can be managed conveniently.
Furthermore, the application allows users to disable login for specific users and hide import/export buttons across models. The tool aims to empower users with comprehensive access control functionalities, offering a seamless user experience without the need for complex technical skills.
