Perfect Inclusion for Your B2B Customers

  • Designed to promote quicker orders and high sales.
  • Help customers easily place bulk orders from Website.

Separate Quick Order Page on the Website

  • Added a separate quick order page on the website.
  • The quick order menu shows website products in list form.

Search Bar Added on Quick/Bulk Order Page

  • Added a quick search bar on the Quick Order page.
  • Use the Quick Order search bar to find desired products.
  • Search products with names, initials, etc.

Add products to Quick/Bulk Order Wishlist

  • The products first go to the Quick Order wishlist.
  • The customer can save products from a wishlist to a shopping list and place bulk orders.
  • The customer can save products from a wishlist to a shopping list and place bulk orders.
  • Alternatively, directly place an order from the wishlist.

Enable Auto-Save for Bulk Order Wishlist

  • The bulk order wishlist is not reset if the customer logs out of his account.
  • The products added to the wishlist are recovered on logging again.
  • The customer can resume from the same point when they log in again.

Create Different Shopping Lists

  • The customer can create and save custom shopping lists in his account.
  • Add products from quick/bulk order wishlists to shopping lists.

View and Edit Shopping Lists

  • The customer can see and edit his shopping lists from his website account.
  • Merge different shopping lists with one click.
  • The Odoo user can see the customers' shopping lists in the Odoo backend.

Quick Order history

  • Displayes the bulk order separately from regular website orders in the backend.

Repeat Previous Quick Orders Again

  • Customers can reorder their bulk orders from their website accounts.
  • Click on the reorder button to place the entire order again.
  • The product list can be edited while reordering the bulk order.

Manage Quick order popup alerts

  • Configure various message popups for customers.
  • Edit informative messages for empty wishlists, product searches, etc.
  • Also, configure messages for empty shopping lists.
