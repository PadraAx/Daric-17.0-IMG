Create Project using Template in Odoo


This app will help you to set your projects as templates, and then create new projects by templates. User can use template to create project from selected template quickly.

Helps to Make Project

Step by Step Process

Process done will step by step.

Create Project From Template

Easy to Create new project from the template.

User can See Project List

User can see all project and new created project list.




Templates Of Projects Menu

Under project > Configuration > Templates Of Projects menu users can create templates for projects.

Set In Progress

User can click on "SET IN PROGRESS" button to set template in progress.

After clicking "Set to In Progress" state will change to "In Progress". Click "Set as Template".

After clicking "Set as Template", the status will change to "Template". To create the same project, click "New Project Based on Template".

Created project from Template

User can see newly created project using project template.

Click on "Reset as Project" state will change to "Set In Progress"

After clicking "Reset as Project" state will change to "New".
