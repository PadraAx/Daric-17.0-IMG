Overview

Purpose of the module is to provide popup features. The module is used as a dependency by other modules. It has no direct visible effect in the system.

This popup function can be invoked within any model, where self represents an object of that model, such as a partner, sale order, invoice, etc.
Example Usage: base_model_attributes.
